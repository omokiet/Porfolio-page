# Portfolio
Cao Võ Tuấn Kiệt (kietcvt) - Trang Web Hồ Sơ Cá Nhân.

## Tổng Quan
Một trang web hồ sơ cá nhân hiệu năng cao, dựa trên kiến trúc Single Page Application, được xây dựng bằng các công nghệ web hiện đại. Tập trung vào giao diện người dùng cao cấp, thiết kế mờ ảo (glassmorphism) và bố cục hiển thị tốt trên mọi thiết bị nhằm giới thiệu kỹ năng, dự án và chứng chỉ nổi bật.

## Ngôn Ngữ và Công Nghệ
- Nền tảng Frontend: React 19, TypeScript
- Trình đóng gói: Vite
- Trình bày giao diện: Tailwind CSS
- Thư viện biểu tượng: Lucide React

## Cấu Trúc Dự Án
- /src/components: Các thành phần giao diện tái sử dụng (Hero, About, Skills, Projects, Certificates, v.v.).
- /src/constants: Dữ liệu cấu hình cốt lõi (danh sách kỹ năng, dữ liệu dự án, thông tin cá nhân) được tách biệt khỏi logic xử lý.
- /src/assets: Các tập tin đa phương tiện và hình ảnh tĩnh.

## Tính Năng Nổi Bật
- Bố cục dạng lưới động (Bento Grid).
- Các thành phần giao diện theo phong cách kính mờ (Glassmorphism).
- Khả năng phản hồi tối ưu trên màn hình di động, máy tính bảng và PC.
- Các hiệu ứng chuyển động mượt mà và hiệu ứng ánh sáng phát quang tinh tế.
- Phân tách rõ ràng giữa cấu trúc dữ liệu (nhóm constant) và giao diện hiển thị lớp trình bày.

## Hướng Dẫn Cài Đặt

1. Sao chép kho lưu trữ:
git clone https://github.com/omokiet/Porfolio-page.git

2. Cài đặt các gói phụ thuộc:
npm install

3. Khởi chạy máy chủ phát triển cục bộ:
npm run dev

4. Xây dựng phiên bản sản xuất:
npm run build

## Lưu Ý Kiến Trúc
- Sử dụng Functional Components và React Hooks.
- Tận dụng sức mạnh tiện ích sâu của Tailwind, tránh làm cồng kềnh mã CSS tùy chỉnh.
- Giao diện mặc định là nền tối với điểm nhấn gradient màu ngọc lục bảo (Emerald).
